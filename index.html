Things to know about JavaScript...

  • + is for addition
  • - is for subtraction
  • / backslash is for divide
  • * is for multiply
  • % is for modulus division

There are a few command words we need to discuss. We have the following command words:

  • document: refers to the current document
  • getElementByID: allows us to grab an element on the webpage based on the id
  • textContent: allows us to change the text content within an element
  • function: allows us to save code for later
  • value: allows us to grab or set the value of a form element
  • let: allows us to create a variable
  • onclick: is an event that occurs when the user clicks something on the page
